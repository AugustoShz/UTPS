# .NET Framework

Exercício feito com único propósito de ingresso no processo de seleção para oportunidade de desenvolvimento na Uppertools Tecnologia da Informação

# Justificativas
Foi utilizado uma classe chamada "Empresa" para guardar as informações obtidas tanto do banco de dados quanto da API para assim diminuir dificuldades na hora de converter os dados para tabela de dados, Json ou objeto.

## Arquitetura
Para uma melhor separação dos códigos, foram criadas varias classes para lidar com cada funcionalidade, como por exemplo as funções que necessitavam de conexão com o banco de dados e funções que necessitavam de conexão com a API.

## Tecnologias

Foram utilizadas como o requisitado C# .NET Framework e como comodidade Windows Form Application. Para o banco de dados foi utilizado o MySQL.

## Dificuldades Gerais

A maior dificuldade foi a integração do C# com o MySQL na hora do tratamento dos dados recebidos por ele, como estava sendo utilizado uma classe própria, foi dificil essa conversão de Dataset para Objeto.

## Conclusão

É um projeto que apesar de simples, traz a sua dificuldade e um maior estudo sobre as tecnologias é necessário para o aperfeiçoamento.
